# 📊 Sales Performance Dashboard
Sales analysis project using Excel, SQL, and Power BI


## 🧠 Project Overview
This project analyzes sales data to uncover insights into revenue, profitability, and regional performance using Excel, SQL, and Power BI.

---

## 🎯 Business Problem
The goal is to understand:
- Which products drive revenue
- Which regions perform best
- Sales trends over time
- Profitability across categories

---

## 🛠 Tools Used
- Excel (Data Cleaning)
- PostgreSQL (Data Analysis)
- Power BI (Visualization)

---

## 📊 Dashboard

## 🔍 Key Insights

- The West region generates the highest revenue
- Technology Category is the most profitable
- Sales experience a decline from 2015-2016 but a change in pattern occured henceforth
- A few products contribute most of the revenue
- Some categories have high sales but low profit margins
